# 🌦️ Python Weather App

A simple desktop weather application built with **Python** and **PyQt5**.  
The application allows users to enter a city name and retrieve the current weather information using a weather API.

This project is mainly created for learning purposes, focusing on:
- GUI development with PyQt5
- API integration in Python
- Handling user input
- Displaying dynamic weather data

---

---

## 🚀 Features

- Enter a city name
- Fetch real-time weather data
- Display:
  - Temperature
  - Weather description
  - Weather emoji/icon
- Simple and clean user interface

---

## 🛠️ Technologies Used

- Python
- PyQt5
- Requests
- Weather API
